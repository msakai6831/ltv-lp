# LP活用ガイド

このLP（ランディングページ）の修正・複製・活用方法について解説します。

## 1. ファイル構成
このフォルダには以下のファイルが含まれています。
- **index.html**: ページの骨組みと文章（テキスト）が入っています。修正は主にこのファイルで行います。
- **style.css**: デザイン（色、配置、フォントなど）が書かれています。
- **script.js**: 動き（アニメーションなど）が書かれています。
- **画像ファイル**: `profile_photo.png` や `slides_preview.png` など。

## 2. 修正方法

### テキスト（文章）を変えたい場合
1. `index.html` をテキストエディタ（メモ帳、テキストエディット、VS Codeなど）で開きます。
2. 変えたい文章を検索し（Ctrl+F または Command+F）、書き換えて保存します。
   - 例: `<h3>さかい美佐</h3>` の中の文字を変更。

### 画像を変えたい場合
1. 使いたい画像をこのフォルダに入れます。
2. `index.html` 内の画像ファイル名を書き換えます。
   - 例: `<img src="profile_photo.png">` の `profile_photo.png` を新しいファイル名に変更。
   - **簡単な方法**: 新しい画像を `profile_photo.png` という名前で保存し、元のファイルを上書きすれば、コードを変えずに画像だけ変わります。

### 構成（セクションの並び順）を変えたい場合
LPは「セクション」というブロックごとの積み重ねでできています。
`index.html` の中にある `<section ...>` から `</section>` までがひとつのブロックです。

1. **順番を変える**:
   動かしたいセクションの `<section>...</section>` の塊を切り取って、別の場所に貼り付ければ、表示順が変わります。
   （例: 「お客様の声」を「料金プラン」の上に移動するなど）

2. **セクションを消す**:
   不要なセクションの `<section>...</section>` の塊を削除すれば、その部分は表示されなくなります。

## 3. 複製して別のLPを作る方法
1. デスクトップにあるこのフォルダ（`GoogleAntigravitytest`）ごとコピー＆ペーストします。
2. 複製されたフォルダの名前をわかりやすい名前（例: `LP_ver2` や `Campaign_2025`）に変更します。
3. 複製したフォルダの中の `index.html` などを自由に編集してください。元のLPはそのまま残ります。

## 4. 公開方法
このファイル一式（HTML, CSS, JS, 画像）を、ご利用のレンタルサーバーやホスティングサービスにアップロードすることで、Web上に公開できます。
